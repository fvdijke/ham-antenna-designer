# HAM Antenna Designer

Practical antenna design tool for HAM radio operators: pick an antenna type and a band, get element lengths, feedpoint, counterpoise/radial sizing, choke/balun advice, and a scaled, build-template-quality drawing.

Design doc: see /office-hours session output (Approach A -> C -> B: vertical-first, then print-and-build PDF tiling, then a data-driven multi-antenna engine).
